# 🚀 场景化速查手册 (Scenario Cheatsheet)

按**解决问题**的思路索引，助你快速定位解决方案。

## 1. 磁盘与存储
- **磁盘满了怎么办？**
  1. `df -h`：查看哪个分区满了。
  2. `ncdu` (或 `du -sh *`)：深入目录，找出占用空间的大文件。
  3. `rm` / `trash`：清理垃圾文件。
  4. `docker system prune`：如果是 Docker 占用过多，清理未使用的镜像和容器。

## 2. 系统性能与进程
- **CPU / 内存飙升怎么排查？**
  1. `top` / `htop` / `btop`：查看整体负载。
  2. 按 `P` (CPU排序) 或 `M` (内存排序)。
  3. `strace -p PID`：查看高占用进程具体在执行什么系统调用（需 root）。
  4. `ps -ef | grep 进程名`：查看进程启动参数。

- **程序卡死了？**
  1. `ps aux | grep 程序名`：找到 PID。
  2. `kill PID`：尝试正常终止。
  3. `kill -9 PID`：强制杀死（最后手段）。

## 3. 网络与连接
- **端口通不通？**
  - `telnet IP Port`
  - `nc -zv IP Port` (更推荐)
- **谁占用了端口？**
  - `ss -tulpn | grep :端口号`
  - `lsof -i :端口号`
- **查看本机公网 IP**
  - `curl ifconfig.me`

## 4. 日志与排错
- **日志太多怎么看报错？**
  - `tail -f log_file | grep -i error --color`：实时监控并高亮 "error"。
  - `less log_file` -> 按 `G` 跳到底部 -> 按 `?error` 向上搜索错误。

## 5. 文件查找
- **忘了文件在哪？**
  - `find . -name "*config*"`：在当前目录找。
  - `locate config.ini`：全盘快速找（基于索引）。
  - `fzf`：交互式模糊搜索。
