# 🌟 现代化工具与技巧 (Modern Tools & Tricks)

> 2024年推荐安装的 Rust/Go 替代工具，提升 10 倍幸福感。

## 1. 现代化替代方案 (Modern Replacements)

| 经典命令 | 现代化替代 (推荐安装) | 优势 |
| :--- | :--- | :--- |
| `ls` | **`exa`** (或 `eza`) | 支持 Git 状态显示、图标、树状视图 (`exa --tree`) |
| `cat` | **`bat`** | 支持语法高亮、自动分页、显示行号、Git 变更标记 |
| `find` | **`fd`** | 语法更简单 (`fd pattern`)，默认忽略 `.gitignore`，速度极快 |
| `grep` | **`rg`** (Ripgrep) | 搜索速度最快、智能过滤垃圾文件 |
| `cd` | **`z`** / **`zoxide`** | 智能跳转，根据访问频率直接跳到目录 (如 `z pro` -> `/home/user/projects`) |
| `du` | **`dust`** | 类似 `ncdu` 但更直观的树状磁盘占用图 |
| `diff` | **`delta`** | 极美观的 Git diff 查看器，支持语法高亮 |
| `top` | **`btop`** / `glances` | 比 `htop` 更酷炫，支持鼠标，显示更多硬件信息 |
| `curl` | **`httpie`** (http) | 输出带高亮的 JSON，语法更人性化 |

## 2. 为什么选择现代化工具？
- **速度更快**：许多工具使用 Rust 编写，性能卓越。
- **默认配置更佳**：无需繁琐配置即可获得语法高亮、Git 集成等功能。
- **用户体验更好**：输出结果更直观，支持图标和颜色，减少视觉疲劳。

> **提示**：这些工具通常可以通过系统的包管理器（如 `apt`, `brew`, `pacman`）或 `cargo` 安装。
