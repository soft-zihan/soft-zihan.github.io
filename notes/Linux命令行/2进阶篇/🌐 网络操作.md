### 1. ping / telnet / nc - 连通性测试
- `ping host`：测试网络连通性 (Packet Internet Groper)
- `telnet host port`：测试特定端口连通性
- `nc -zv host port`：更现代的端口测试工具 (Netcat)

### 2. ip / ifconfig / hostname - 地址与主机名
- `ip addr`：显示所有网卡IP地址 (现代标准)
- `ifconfig`：显示网卡信息 (旧版, interface configuration)
- `hostname`：查看或临时设置当前主机名
- `ip route`：显示路由表信息

### 3. netstat / ss / lsof - 端口与连接监控
- `ss -tulpn`：查看所有监听端口及对应进程 (推荐, Socket Statistics)
- `netstat -tulpn`：经典端口占用查看工具 (在某些发行版中已逐渐被 `ss` 替代)
- `lsof -i :端口号`：查看特定端口被哪个进程占用 (List Open Files)
- `sudo nethogs`：按进程查看实时网速占用

### 4. curl / wget - 网络请求与下载
- `curl -I URL`：仅查看HTTP响应头
- `curl -L URL`：跟随HTTP重定向
- `curl -O URL`：下载文件并保持原名
- `curl -X POST -H "Content-Type: application/json" -d '{"key":"value"}' URL`：发送POST请求
- `wget -c URL`：断点续传下载

### 5. ssh / scp / rsync - 远程操作
- `ssh user@host`：通过SSH远程登录 (Secure Shell)
- `scp file user@host:/path`：远程复制文件 (Secure Copy, 已有些过时)
- `rsync -avz source/ user@host:/path`：更高效的增量文件同步 (Remote Sync)

### 6. nslookup / dig - 域名解析
- `nslookup domain`：查询域名解析记录 (Name Server Lookup)
- `dig domain`：更详细的DNS查询 (Domain Information Groper)
